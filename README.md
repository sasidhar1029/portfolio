# Sasidhar Reddy — Portfolio

A personal portfolio website built with **React.js**, HTML5, CSS3, and JavaScript.

## Tech Stack
- React 18
- CSS3 (CSS Variables, Flexbox, Grid)
- JetBrains Mono + Space Grotesk + Inter (Google Fonts)

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm start

# Build for production
npm run build
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

```
src/
  components/
    Navbar.js / Navbar.css
    Hero.js / Hero.css
    About.js / About.css
    Skills.js / Skills.css
    Projects.js / Projects.css
    Experience.js / Experience.css
    Contact.js / Contact.css
    Footer.js / Footer.css
  App.js / App.css
  index.js / index.css
public/
  index.html
```

## Features
- Typewriter animation cycling through job roles
- Smooth scroll navigation
- Responsive design (mobile-friendly)
- Hover micro-interactions on all cards
- Click-to-copy email button
- Reduced motion support

## Customization
- Update **project links** in `src/components/Projects.js`
- Add your **GitHub URL** in `Contact.js`
- Update **project details** as you build more work
